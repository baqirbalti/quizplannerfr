This is the AI Skill Bridge Frontend (Next.js + Tailwind).

## Getting Started

First, copy env and run the development server:

```bash
copy .env.local.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Set `NEXT_PUBLIC_API_BASE` in `.env.local` to your backend URL (default `http://localhost:8000`).

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load Inter.
